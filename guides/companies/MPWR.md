## Navigation Instructions

1. Fetch job listings from the Greenhouse API or navigate to the board URL.
2. Click on a job listing.
3. Click "Apply for this job" to load the application form.
4. Fill text fields: First Name, Last Name, Email, Phone.
5. Upload resume via the Attach button.
6. Fill any combobox fields using the Click-Snapshot-Click pattern.
7. Complete custom questions (work authorization, sponsorship).
8. Fill EEO fields (default to "Decline").
9. Submit the application.
